# 👤 樂行童軍成員使用教學 (Rover Member)

> 你只需關心自己的進度，其他由團管理委員會處理

## 登入後你會見到
- **📊 我的進度**：預設只顯示你自己的7項考驗完成度
- **⭐ 其他獎章**：已併入我的進度最底，顯示急救、AYP金章、寰宇童軍章等
- **📝 待批**：查看你提交的申請是否已批
- **🖨️ 表格列印**：自動填PT/21(樂行童軍獎章)及PT/22(貝登堡獎章)官方格式，記得雙面列印
- **📚 資料庫**：保護兒童、安全指引、第5版訓練綱要
- **❓ 教學**：按角色分開，此文件

## 常見操作
### 如何申請完成？
1. 我的進度找到該項目（例如童軍技能）
2. 按📝申請，選日期+證據(例如訓練班證書)
3. 提交，團管理委員會及領袖會在待批看到
4. 批准後進度自動更新

### 私隱
成員預設只看自己，若旅團開放互相查看，團長可在用戶管理開啟。
